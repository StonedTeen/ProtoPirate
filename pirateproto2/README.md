## **Installation & Usage**

### Requirements
- Flipper Zero with latest firmware
- External CC1101 module (recommended for best performance)
- Sufficient storage space for signal files

### Installation
1. Download the latest PirateProto 2 .fap file
2. Copy to your Flipper Zero's apps folder
3. Launch from the Sub-GHz applications menu

### Basic Usage
1. **Signal Capture**: Use Receiver mode to capture signals
2. **Analysis**: Use Signal Analyzer to examine captured signals
3. **Learning**: Use Protocol Learner to understand new protocols
4. **Transmission**: Use Emulate mode to transmit signals
5. **Management**: Use Saved Signals to organize your collection
